# Makeme

A simple Makefile template to kickstart your projects.

## Quick Start

Download the Makefile to your current directory:

```bash
curl -O https://raw.githubusercontent.com/cmsheehan/makeme/main/Makefile
```

## Optional: Create an Alias

Add a `makeme` command to quickly download the Makefile anywhere:

**Zsh:**
```bash
echo 'alias makeme="curl -O https://raw.githubusercontent.com/cmsheehan/makeme/main/Makefile"' >> ~/.zshrc
source ~/.zshrc
```

**Bash:**
```bash
echo 'alias makeme="curl -O https://raw.githubusercontent.com/cmsheehan/makeme/main/Makefile"' >> ~/.bashrc
source ~/.bashrc
```

Then just run `makeme` in any project directory.
